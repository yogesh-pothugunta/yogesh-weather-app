# 🌦️ Yogesh Weather App

A simple and responsive weather web application that shows real-time weather details using city name.

## 🚀 Live Demo
https://yogesh-weather-app.onrender.com

## 💻 GitHub Repository
https://github.com/yogesh-pothugunta/yogesh-weather-app

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- REST API

## ✨ Features
- Search weather by city name
- Real-time temperature and weather data
- Clean and responsive UI
- Deployed on cloud (Render)
